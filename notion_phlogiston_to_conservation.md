# 불꽃 뒤에 숨겨진 진실 — 플로지스톤에서 질량 보존까지

> "과학은 옳은 답을 쌓아가는 것이 아니라, 더 나은 질문을 찾아가는 과정이다."

---

## 1. 출발점: 연소란 무엇인가?

나무가 타면 재만 남는다. 쇠가 녹슬면 무거워진다.
금속을 가열하면 어떤 것은 가벼워지고 어떤 것은 무거워진다.

18세기 화학자들이 가장 집착했던 질문:

> **"불꽃이 타오를 때, 무엇이 사라지는 것인가?"**

---

## 2. 플로지스톤 설 — 한 시대의 정답

### 이론의 탄생

| | |
|---|---|
| **제안자** | 게오르크 에른스트 슈탈 (Georg Ernst Stahl, 1703) |
| **기반** | 요한 베허(Becher)의 "기름 원소" 가설을 발전 |
| **핵심 주장** | 모든 가연성 물질에는 **플로지스톤(Phlogiston)** 이라는 불 원소가 들어 있다 |

### 플로지스톤 설의 설명 방식

```
연소 전 물질 = 재(ash) + 플로지스톤
연소 시       → 플로지스톤이 공기 중으로 방출
연소 후       → 재만 남음
```

**나무가 타면 재만 남는 이유?** → 플로지스톤이 빠져나갔으므로

**금속(마그네슘)을 태우면 왜 더 무거워지지?** → 플로지스톤이 "음의 질량"을 가진다고 임시방편 수정

### 당시 과학계의 반응

- 연소, 호흡, 산화를 하나의 개념으로 설명 → **설득력 있는 통합 이론**
- 프리스틀리(산소 발견자)조차 죽을 때까지 플로지스톤 설을 지지
- 약 80년간 유럽 화학계의 **공식 패러다임**으로 군림

## 3. 라부아지에의 실험 — 패러다임이 무너지다

### 인물 소개

> **앙투안 라부아지에 (Antoine-Laurent de Lavoisier, 1743–1794)**
> 프랑스 화학자. "근대 화학의 아버지"
> 프랑스 혁명 시기 세금 징수관으로 일하다 단두대에서 처형.

### 핵심 실험 1 — 밀폐 용기 속 주석 연소 (1770년대)

**실험 설계의 혁명성:** 라부아지에는 반응 전후에 **용기 전체**를 저울로 쟀다.

```
[실험 과정]
1. 주석(Sn)을 밀폐 유리 용기 안에 넣는다
2. 용기 전체의 질량을 측정 → M₁
3. 가열하여 주석을 연소
4. 용기를 냉각 후 전체 질량 재측정 → M₂

[결과]
M₁ = M₂  (밀폐 용기 전체 질량은 불변)
단, 뚜껑을 열면 공기가 들어오며 질량 증가
```

**해석:** 주석이 산화될 때 증가하는 질량 = 용기 안 공기에서 흡수한 성분의 질량

### 핵심 실험 2 — 수은 연소와 "탈플로지스톤 공기"

```
[실험]
수은(Hg)을 밀폐 용기 안에서 가열 → 붉은 산화수은(HgO) 생성

[관찰]
- 용기 안 기체 부피 감소 (공기가 소비됨)
- 남은 기체: 생물이 살지 못하고 촛불도 꺼짐

[역반응]
산화수은(HgO)을 고온으로 재가열
→ 다시 수은 + 기체 방출
→ 방출된 기체: 촛불이 더 밝게 타오름
```

프리스틀리는 이 기체를 "플로지스톤이 빠진 공기"라 불렀다.
라부아지에는 달랐다:

> "이것은 새로운 원소다. **산소(Oxygène)**라 명명한다."

### 라부아지에의 결론

연소 = **산소와의 결합** (플로지스톤 방출이 아니라 산소 흡수)

```
플로지스톤 설:  금속 → 재 + 플로지스톤 (방출)
라부아지에:     금속 + 산소 → 금속산화물 (결합)
```

## 4. 질량 보존의 법칙 정립

### 법칙의 선언 (1789년, "화학 원론")

> **화학 반응이 일어나기 전후에 물질의 전체 질량은 변하지 않는다.**

```
반응물의 총질량 = 생성물의 총질량
```

### 왜 "발견"이 아니라 "측정"인가

라부아지에 이전에도 질량이 보존된다는 직관은 있었다.
라부아지에의 공헌은 **정밀한 저울**로 이를 **정량적으로 증명**한 것.

| 항목 | 플로지스톤 시대 | 라부아지에 이후 |
|---|---|---|
| 연소 설명 | 불 원소 방출 | 산소와 결합 |
| 질량 변화 | 설명 불가 (음의 질량 임시방편) | 전후 총질량 보존 |
| 측정 방법 | 반응물 단독 측정 | 밀폐계 전체 측정 |
| 기체 인식 | 공기는 단일 물질 | 공기는 혼합물 (N₂ + O₂ 등) |

### 실제 수치 예시

**수소 연소 반응:**
```
2H₂  +  O₂  →  2H₂O
4g  +  32g  =  36g    (질량 보존)
```

**철 산화:**
```
4Fe  +  3O₂  →  2Fe₂O₃
224g + 96g  =  320g   (질량 보존)
철이 녹슬면 무거워지는 이유 = 산소가 결합했기 때문)
```

## 5. 과학의 본성(NOS)과의 연결

### 이 이야기에서 읽어야 할 것들

#### NOS 1 — 과학적 지식은 잠정적이다

플로지스톤 설은 **틀린 이론이 아니었다. 당시의 최선이었다.**
80년간의 정밀한 관찰과 실험을 설명했고, 학계가 수용했다.

> 오늘날 우리가 당연하게 여기는 지식도 미래에는 수정될 수 있다.
> 과학은 "영원한 진리"가 아닌 **"지금까지 가장 잘 설명하는 모델"** 이다.

#### NOS 2 — 실험은 이론을 반증할 수 있다

라부아지에가 플로지스톤 설을 무너뜨린 무기는 **저울**이었다.
데이터가 이론과 맞지 않을 때, 두 가지 선택이 있다:

- A) 이론을 지키기 위해 데이터를 해석한다 (플로지스톤 = 음의 질량)
- B) 데이터를 존중하고 이론을 수정하거나 폐기한다

라부아지에는 B를 선택했다. 이것이 **과학적 태도**다.

#### NOS 3 — 과학은 사회·문화적 맥락 안에 있다

프리스틀리는 산소를 먼저 발견했지만 플로지스톤 설의 틀 안에서 해석했다.
라부아지에는 같은 현상을 **다른 개념 틀**로 바라봐서 혁명을 일으켰다.

> 어떤 질문을 하느냐가, 어떤 답을 발견하느냐를 결정한다.

#### NOS 4 — 과학자도 인간이다

- 프리스틀리: 끝까지 플로지스톤 설을 고수 (심리적 저항)
- 라부아지에: 귀족 세금 징수관 → 단두대 처형 (사회적 조건)
- 과학적 발견은 순수한 지적 활동만이 아니라 시대, 정치, 경제와 얽혀 있다.

```
    ↓ 화학 반응식으로 표현
반응물 총질량 = 생성물 총질량
    ↓ 과학의 본성

```
