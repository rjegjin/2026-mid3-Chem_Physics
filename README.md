# 🧪 중3 과학 실험실: 화학 반응의 규칙

중학교 3학년 과학 '화학 반응의 규칙' 단원을 위한 인터랙티브 강의 자료입니다. 학생들의 흥미를 유발하고 시각적으로 이해를 돕기 위해 프레젠테이션 스타일로 제작되었습니다.

## 🌟 주요 특징

- **인터랙티브 슬라이드:** 클릭 및 키보드 조작을 통한 순차적 정보 공개 (Reveal)
- **시각적 분자 모델:** CSS로 구현된 2D/3D 스타일의 분자 모델링
- **반응형 디자인:** PC, 태블릿, 모바일 기기 어디서나 최적화된 화면 제공
- **수식 지원:** MathJax를 활용한 정확하고 깔끔한 화학 반응식 표기
- **학습 확인 퀴즈:** 각 강의 끝에 배치된 인터랙티브 퀴즈와 즉각적인 피드백
- **눈이 편한 디자인:** 장시간 학습에도 피로도가 적은 색상 배합
