Case Study 01

앙금 생성 반응과 질량

액체와 액체가 만나 고체(앙금)가 생길 때의 변화

반응물 (Before)

염화 나트륨 + 질산 은

\(NaCl + AgNO_3\)

생성물 (After)

염화 은(↓) + 질산 나트륨

\(AgCl \downarrow + NaNO_3\)

실험 결과 분석: 앙금

반응 전 질량

150.45 g
＝

150.45 g

심화: 나무의 연소 (질량 감소?)

Open System Analysis

날아간 기체 범인들 찾기 🔍

나무 속의 탄소(\(C\))와 수소(\(H\))가 산소와 만나 이산화 탄소(\(CO_2\))와 수증기(\(H_2O\))가 됩니다.

이들은 **기체** 상태이므로 공기 중으로 날아가 버립니다. 남은 **재**의 무게만 재면 질량이 줄어든 것처럼 보일 뿐입니다.

\(나무 + 산소 = 재 + \color{red}{CO_2 \uparrow + H_2O \uparrow}\)

심화: 강철 솜의 연소 (질량 증가?)

Open System Analysis

새롭게 합류한 산소 대원들 ⚓

강철 솜(철, \(Fe\))이 타면 공기 중의 산소(\(O_2\))가 철 원자와 강하게 결합하여 고체 상태의 산화 철이 됩니다.

외부(공기)에서 들어온 산소의 질량이 고체 속에 **박혔기 때문에** 전체 질량은 처음보다 늘어나게 됩니다.

\(철(Fe) + \color{blue}{산소(O_2)} = 산화 철\)
