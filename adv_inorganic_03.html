제 3강: 분자의 춤을 해독하다

분자의 진동 (Molecular Vibrations)

분자 내의 원자들은 정지해 있지 않고, 스프링으로 연결된 질량체처럼 끊임없이 진동합니다.

기준 진동 모드 (Normal Modes)

복잡해 보이는 분자의 전체 진동은, 각 원자가 동일한 주파수와 위상으로 단순 조화 진동을 하는 독립적인 기본 진동들의 선형 결합으로 표현됩니다.

"군론(Group Theory)은 복잡한 미분 방정식을 풀지 않고도, 대칭성만으로 이 기준 진동 모드들의 형태와 분광학적 활성도를 완벽하게 예측하게 해줍니다."

운동의 자유도 (Degrees of Freedom)

$N$개의 원자로 이루어진 분자는 각 원자당 $x, y, z$ 축으로 이동할 수 있으므로, 총 $3N$개의 자유도를 가집니다.

비선형 분자 (Non-linear)

  • 전체 운동: $3N$
  • 병진 운동 (Translation): $3$ ($x, y, z$)
  • 회전 운동 (Rotation): $3$ ($R_x, R_y, R_z$)
  • 진동 자유도: $3N - 6$

선형 분자 (Linear)

  • 전체 운동: $3N$
  • 병진 운동 (Translation): $3$ ($x, y, z$)
  • 회전 운동 (Rotation): $2$ (분자축 중심 회전은 무의미)
  • 진동 자유도: $3N - 5$

진동 분석 1단계: $\Gamma_{3N}$ 도출

분자의 3N 카테시안 좌표 전체를 기저로 삼아, 점군의 조작들에 대한 가약 표현(Reducible Representation)을 구합니다.

\[ \chi_{3N}(R) = N_{unshifted} imes \chi_{xyz}(R) \]
  • $N_{unshifted}$: 대칭 조작 $R$을 가했을 때 위치가 변하지 않는 원자의 수. (위치가 변한 원자는 대각합에 기여하지 않음)
  • $\chi_{xyz}(R)$: 단일 원자의 $(x,y,z)$ 좌표가 조작 $R$에 의해 변환될 때의 지표.

실습 1: $H_2O$ ($C_{2v}$)의 $\Gamma_{3N}$

물 분자($N=3$)의 총 자유도는 $3(3) = 9$ 입니다.

$C_{2v}$$E$$C_2$$\sigma_v(xz)$$\sigma_v'(yz)$
$N_{unshifted}$31 (O)3 (O, H, H)1 (O)
$\chi_{xyz}$3-111
$\Gamma_{3N}$9-131

($\sigma_v(xz)$는 분자 평면이므로 3원자 모두 이동하지 않고, $\sigma_v'(yz)$는 H원자 둘을 교환하므로 O원자 1개만 제자리에 남습니다.)

실습 2: 약분 공식을 통한 분해

얻어진 $\Gamma_{3N} = (9, -1, 3, 1)$ 을 $C_{2v}$ 기약 표현들로 분해합니다.

\[ n_i = \frac{1}{h} \sum_R g_c \chi_i(R) \chi_{3N}(R) \]
$n_{A_1} = \frac{1}{4}[ (1)(9) + (1)(-1) + (1)(3) + (1)(1) ] = \frac{12}{4} = 3$
$n_{A_2} = \frac{1}{4}[ (1)(9) + (1)(-1) + (-1)(3) + (-1)(1) ] = \frac{4}{4} = 1$
$n_{B_1} = \frac{1}{4}[ (1)(9) + (-1)(-1) + (1)(3) + (-1)(1) ] = \frac{12}{4} = 3$
$n_{B_2} = \frac{1}{4}[ (1)(9) + (-1)(-1) + (-1)(3) + (1)(1) ] = \frac{8}{4} = 2$
\[ \Gamma_{3N} = 3A_1 + A_2 + 3B_1 + 2B_2 \]

실습 3: 순수 진동 모드($\Gamma_{vib}$) 추출

전체 운동 $\Gamma_{3N}$ 에는 병진 운동(Translation)과 회전 운동(Rotation)이 포함되어 있습니다. 지표표를 보고 이들을 빼줍니다.

  • 병진 운동 ($\Gamma_{trans}$): 지표표에서 $x, y, z$ 에 해당하는 대칭종.
    $\implies A_1(z) + B_1(x) + B_2(y)$
  • 회전 운동 ($\Gamma_{rot}$): 지표표에서 $R_x, R_y, R_z$ 에 해당하는 대칭종.
    $\implies A_2(R_z) + B_1(R_y) + B_2(R_x)$
\begin{align} \Gamma_{vib} &= \Gamma_{3N} - \Gamma_{trans} - \Gamma_{rot} &= (3A_1 + A_2 + 3B_1 + 2B_2) - (A_1 + B_1 + B_2) - (A_2 + B_1 + B_2) &= \mathbf{2A_1 + B_1} \end{align}

(비선형 3원자 분자이므로 진동 자유도는 $3(3)-6 = 3$개. $2A_1 + B_1$ 으로 정확히 3개가 나옵니다.)

$H_2O$의 3가지 기준 진동 모드

수학적으로 유도해낸 $2A_1 + B_1$ 이 실제로 어떤 움직임인지 살펴봅시다.

Symmetric Stretch

두 O-H 결합이 동시에 늘어나고 줄어듦. 모든 대칭 조작에 불변.

대칭종: $A_1$

Bending

결합각이 커졌다 작아짐. 역시 대칭성이 깨지지 않음.

대칭종: $A_1$

Asymmetric Stretch

한 결합이 늘어날 때 다른 쪽은 줄어듦. $C_2$ 회전 시 부호(위상)가 반전됨.

대칭종: $B_1$

분광학적 선택 규칙 (Selection Rules)

어떤 진동 모드가 적외선(IR)을 흡수하거나 라만(Raman) 산란을 일으키는지 군론으로 완벽히 결정할 수 있습니다.

1. 적외선 (IR) 활성 조건

진동에 의해 분자의 쌍극자 모멘트(Dipole Moment)가 변해야 합니다.

$\implies$ 진동 모드의 대칭종이 지표표의 선형 함수 $(x, y, z)$ 와 같아야 함.

2. 라만 (Raman) 활성 조건

진동에 의해 분자의 편극률(Polarizability)이 변해야 합니다.

$\implies$ 진동 모드의 대칭종이 지표표의 이차 함수 $(x^2, y^2, z^2, xy, yz, zx)$ 와 같아야 함.

$H_2O$ 의 스펙트럼 예측

앞서 구한 $\Gamma_{vib} = 2A_1 + B_1$ 을 지표표와 대조해봅니다.

  • $A_1$ 모드

    지표표 확인: $z$ (IR), $x^2, y^2, z^2$ (Raman)

    결과: IR 활성, Raman 활성 모두 가능 (Active)

  • $B_1$ 모드

    지표표 확인: $x$ (IR), $xz$ (Raman)

    결과: IR 활성, Raman 활성 모두 가능 (Active)

"물 분자의 3가지 진동 모드는 모두 IR 흡수 스펙트럼과 라만 스펙트럼에서 관측됩니다."

상호 배타의 원리 (Rule of Mutual Exclusion)

반전 중심($i$, Inversion Center)을 가진 분자들에 적용되는 절대적인 분광학 법칙입니다.

!

반전 중심을 가지는 분자(예: $CO_2, SF_6, Benzene$)에서,
IR 활성인 진동 모드는 Raman 비활성이며,
Raman 활성인 진동 모드는 IR 비활성이다.
(동시에 둘 다 활성인 모드는 존재하지 않는다.)

군론적 이유: 반전 중심($i$)이 있으면 대칭종이 $g$(gerade, 대칭)와 $u$(ungerade, 반대칭)로 나뉩니다.
선형 함수($x,y,z$)는 항상 $u$ 대칭성을, 이차 함수($x^2, xy$ 등)는 항상 $g$ 대칭성을 가지므로 교집합이 생길 수 없습니다.

내부 좌표계 (Internal Coordinates) 활용

복잡한 분자에서 3N 전체를 계산하는 것은 너무 번거롭습니다. 특정 결합의 늘어남(Stretch)만 알고 싶다면, 그 결합 벡터(Bond Vectors)만을 기저로 삼을 수 있습니다.

$\Gamma_{stretch}$ 구하기

중심 원자와 연결된 $k$개의 결합 화살표를 기저로 삼습니다.

$\chi_{stretch}(R)$ = 이동하지 않은 결합(Bond)의 개수

이 가약 표현을 약분하면 신축 진동(Stretching modes)의 대칭종만 바로 얻을 수 있습니다.

$\Gamma_{bend}$ 구하기

결합각(Bond angles)을 기저로 삼아 변환 행렬을 구합니다.

$\Gamma_{vib} = \Gamma_{stretch} + \Gamma_{bend}$

전체 진동 모드를 물리적 의미를 가진 신축/굽힘 모드로 세분화하여 분석할 수 있습니다.

Lab: 진동 분광학 활성 판별기

도출된 진동 대칭종을 입력하여 IR/Raman 활성 여부를 확인합니다. (기본 $C_{2v}$ 기준)

대칭종을 선택하면 분광학적 활성도가 표시됩니다.

결론 및 요약

1. 진동 자유도: 비선형 분자는 $3N-6$, 선형 분자는 $3N-5$ 개의 독립적인 기준 진동 모드를 가집니다.
2. $\Gamma_{3N}$ 의 분해: 원자 좌표 전체를 변환시켜 $\Gamma_{trans}$ 와 $\Gamma_{rot}$ 을 제거하면 순수 진동 모드 $\Gamma_{vib}$ 만 남습니다.
3. 선택 규칙: 선형 함수($x,y,z$)와 매칭되면 IR 활성, 이차 함수와 매칭되면 Raman 활성입니다.
4. 상호 배타의 원리: 반전 중심($i$)이 있는 분자는 동일한 모드가 IR과 Raman에서 동시에 관측되지 않습니다.
